### BIENVENIDO A LOCALSMARTA ###
Es un sistema web desarrollado como proyecto de títilo para apoyar la operación de las tiendas de retail. Permite gestionar producto, tareas, incentivos para trabajadores y apoya las decisiones por medio de alertas automáticas relacionadas con las tareas y el stock de productos.

## TECNOLOGÍAS USADAS

# PARA EJECUCIÓN DEL BACKEND
Abrir la terminal en carpeta backend
Instalar dependencias: npm install
Ejecutar el servidor: node app.js
(Servidor disponible en http://localhost:3000)

# PARA ABRIR EL FRONTEND
Abrir la carpeta frontend utilizando live server en visual studio code.

# BASE DE DATOS
La base de datos incluye las siguientes tablas principales:
usuario
producto
tarea
historial_stock
incentivo
reporte

# PARA IMPORTAR LA BBDD
Abrir MySQL Workbench
Crear esquema llamado localsmart
Ejecutar el script incluido
